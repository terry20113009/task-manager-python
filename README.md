# 📝 Task Manager in Python

A simple command-line task manager built with Python.
This project allows you to add, view, and delete tasks, and it saves them automatically in a file.

---

## 🚀 Features

* Add new tasks
* View all tasks
* Delete tasks by number
* Save tasks in a file (`tareas.txt`)

---

## 🛠️ Technologies

* Python 3

---

## ▶️ How to Run

1. Clone this repository:

```
git clone https://github.com/your-username/task-manager-python.git
```

2. Go to the project folder:

```
cd task-manager-python
```

3. Run the program:

```
python main.py
```

---

## 📁 Project Structure

```
task-manager-python/
│
├── main.py       # Main program
├── tareas.txt    # File where tasks are saved
└── README.md     # Project documentation
```
